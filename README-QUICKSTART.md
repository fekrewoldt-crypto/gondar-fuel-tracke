# ⛽ Gondar Auto Services - Quick Start

## 🚀 One-Click Launch (Mac)

**Double-click:** `START-DEMO.command`

Then open your browser to: **http://localhost:3000**

---

## Manual Start (Any OS)

```bash
# Open terminal in this folder, then:
node server.js
```

Then open: **http://localhost:3000**

---

---

## Features

✅ **Interactive Map** - Centered on Gondar with Azezo corridor included  
✅ **Filter by Type** - Toggle between fuel, mechanics, tire shops, car wash  
✅ **Color-Coded Status** - Green (Available/Open), Yellow (Low/Busy), Red (Empty/Closed)  
✅ **Click Markers** - See name, status, services, phone number, last update  
✅ **Submit Reports** - Update status for any location  
✅ **Real-Time Updates** - Map refreshes after each submission  
✅ **Stats Dashboard** - See counts for each service type  
✅ **Mobile Responsive** - Works on phones and desktops

## Science Fair Demo Script

1. **Open the app** - Double-click `START-DEMO.command`
2. **Show the map** - Point out the coverage area (Gondar + Azezo)
3. **Use filters** - Click "Fuel", "Mechanic", "Tire" to show different services
4. **Click a marker** - Show the popup with details
5. **Submit a report** - Change a station's status and watch it update
6. **Show stats** - Explain the 28 locations across 4 categories
